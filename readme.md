# git-interactive

**Work in progress**

## Installation

**Homebrew (macOS)**

```
brew tap josa42/homebrew-git-tools
brew install git-interactive
```

**Other**

```
go get github.com/josa42/git-interactive
```

## Usage

```
Usage:
  git-interactive commit
  git-interactive rebase

Options:
  -h --help          Show this screen.
  --version          Show version.
```
